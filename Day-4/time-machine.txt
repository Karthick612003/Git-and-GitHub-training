🛠️ DAY 4: The Time Machine & The Auditor
Today’s focus was on Recoverability and Precision. You moved from just saving files to being able to jump between different versions of your project.

1. The History Book (git log)
Before you can travel back, you need the "Address" of your past.

Command: git log --oneline

What you learned: Every commit has a unique 7-character Hash (like 6ac830c). This is the ID for that specific moment in time.

2. The Auditor (git diff)
You learned how to see exactly what changed before you save it.

Command: git diff

What you learned: * + Green lines: New additions or edits.

- Red lines: Deletions or old versions of edited lines.

Advanced Tip: Use git diff <hash> <filename> to see how a file has changed since Day 1.

3. The Time Traveler (git checkout)
The most powerful (and slightly scary) command you learned today.

Jump to the Past: git checkout <hash>

Return to Today: git checkout main

The "Folder Lock" Lesson: You discovered that Windows will block Git from changing history if you have Notepad or File Explorer open in that folder. You must close them first!

🏁 "End of Class" Daily Sync
Once you have pasted this into your file, run your final sequence to lock it in:

Stage: git add .

Commit: git commit -m "Day 4: Mastered Git Log, Diff, and Checkout (The Time Machine)"

Push: git push

🔄 The Git WorkflowThese are the core commands I use daily to track my project:CommandActiongit initInitializes a new Git repository.git statusChecks the status of files (What's modified? What's staged?).
    git add .Adds all changes to the "Staging Area."
    git commit -m "message"Saves the current snapshot with a note.
    git log - Views the commit history.

⏱ Version Control "Time Machine"If I need to fix a mistake:
    git checkout -- <filename>: Reverts changes to a file back to the last commit.
    git reset --soft HEAD~1: Undoes the last commit but keeps the files in the folder.
    git reset --hard HEAD~1: Undoes the last commit and deletes all changes in that commit.

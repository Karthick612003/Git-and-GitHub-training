🚀 Day 2: The GitHub Bridge
I have successfully connected my local "Physical Vault" (PC) to my "Digital Vault" (GitHub). My code is now safely backed up in the cloud!

🔗 Connecting Local to Cloud (The Bridge)
Done once to initialize the connection:

Bash
git remote add origin https://github.com/Karthick612003/300-days-automation.git
git remote -v  # Verifies the connection to the URL
📦 The Daily "Save & Sync" Flow
This is the rhythm I follow to keep my local and cloud repositories in sync:

Pack: git add . (Stages all changes)

Label: git commit -m "Your message" (Saves the snapshot)

Sync (Pull): git pull origin main --rebase (Updates local with any cloud changes)

Ship (Push): git push -u origin main (Sends work to GitHub)

🛠 The "Pro" Toolbox
Fixing Mistakes (Amend): If I realize I made a typo after committing, I can fix it without creating a new commit:

Bash
git add .
git commit --amend --no-edit
git push origin main --force
Note: Use --force carefully! It tells GitHub to overwrite its history with mine.

The "q" Key: If the git log screen gets stuck, I just press q to return to the prompt.
